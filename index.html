Welcome to Custom Video Controls

You can you can even customize the player.

Add CvC to your Website today.

Installation

Put this code between the <head> and </head>:

<script src="https://cvidcontrols.github.io/v/1.0.0/code/CvC.js" type="text/javascript"></script>

To check whether CvC has installed successfuly, type this between the <body> and </body>:

<script type="text/javascript">
  alert(ready);
</script>

You should see an alert saying:
Yes! I am always ready! Thank you for using our CvC ðŸ¤—.
after this, you should remove the alert script.

Usage

Adding the video: 
var yourVideo = {
  append: 'your-video-container',
  src: 'videos/your-video.mp4',
  id: 'id-of-your-video',
  class: 'class-of-your-video',
  width: '800px',
  thumb: 'your-video-thumbnail.png',
  autoplay: false,
  muted: false,
  vol: 1.0,
  loop: true,
  context: 'default'
}

newCvC(yourVideo);
